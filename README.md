# Finance

Activate a local virtual environment and install requirements
```bash
python3 -m venv <venv-name>
source <venv-name>/bin/activate
pip install -r requirements.txt
```

To deactivate `$ deactivate`.

To run tests
```bash
pytest -v # i.e. verbose
```
